# Business Sales Performance Analytics

**Internship:** Data Science & Analytics – Task 1

## Overview
This project analyzes retail sales data to uncover key business insights such as revenue trends, top-performance products, and country-level sales performance.

The objective is to transform raw transactional data into meaningful insights that can help businesses make informed decisions.

## Dataset
Online Retail Dataset containing e-commerce transactions including product information, quantity sold, prices, and customer locations.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Github

## Data Analysis Process
1. Data cleaning and preparation
2. Exploratory data analysis (EDA)
3. Revenue trend analysis
4. Product performance analysis
5. Country-level sales analysis
6. Visualization of business insights

## Key Insights
- Revenue trends show fluctuations across months, highlighting seasonal demand.
- A small number of countries contribute the majority of total revenue.
- Certain products dominate sales volume and revenue generation.
- Product demand varies significantly across regions.

## Visualizations
The following visualizations were created to analyze business performance:

- Monthly Revenue Trend
- Top Products by Revenue
- Top Countries by Revenue
- Top Products by Quantity Sold
